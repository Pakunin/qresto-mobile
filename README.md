# **QResto – Android App**


###  Overview  
QResto Android App is for students to use the QResto Attendance System.  
Students can log in, scan classroom QR codes generated by faculty, and mark their attendance through **face verification** using **Google ML Kit**.

---

###  Features
-  **Student Login** via secure backend authentication  
-  **QR Code Scanner** for course-based attendance  
-  **Face Detection** using Google ML Kit to prevent proxy attendance  
-  **Cloud Integration** with Spring Boot backend and MongoDB  
-  **View Attendance History** (per student and per course)

---

###  Tech Stack
| Layer | Technology |
|-------|-------------|
| UI | XML Layout + Material Design |
| Logic | Java |
| Face Detection | Google ML Kit |
| QR Scanning | ZXing Library |
| Networking | OkHttp (REST API Calls) |
| Backend | Spring Boot (Hosted on Railway) |
| Database | MongoDB Atlas |

---

###  Architecture Flow
Login → Dashboard → Scan QR → Face Detection → Attendance Marked → View Records
